# 기능 요구사항

게임 목록 페이지에 적용할 핵심 기능입니다.

---

## 라이트 / 다크 모드

- **라이트 모드**: 기본(주간) 테마
- **다크 모드**: 저조도 환경용 테마
- **구현 방식**:
  - `prefers-color-scheme` 미디어 쿼리로 시스템 설정 자동 감지
  - 사용자 수동 전환 버튼 (localStorage 저장)
  - CSS 변수 기반 테마 전환
- **적용 범위**: 배경, 텍스트, 카드, 버튼, 광고 영역 등 전체 UI

---

## 다국어 (i18n)

- **지원 언어**: 한국어, 영어, 중국어, 일본어 (기존 ChipGames와 동일)
- **적용 대상**:
  - 페이지 제목, 설명
  - 게임명, 게임 카드 레이블
  - UI 라벨 (다크 모드 전환, 언어 선택 등)
  - 푸터, 법적 페이지 링크
- **구현 방식**: `lang/` 폴더의 번역 파일 + 언어 선택기
- **참고**: `delete/lang/` 구조를 참고하여 목록 페이지용으로 축소

---

## Google AdSense

- **Ads.txt**: 기존 `pub-2533613198240039` 유지
- **광고 위치**: 상단 배너, 게임 목록 사이 또는 하단
- **주의사항**:
  - HTTPS 필수
  - 로컬 개발 환경에서는 광고 미표시 (정상)
  - 광고 영역은 라이트/다크 모드에 맞게 테두리·배경 처리
