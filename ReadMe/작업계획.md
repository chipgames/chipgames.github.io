# 작업 계획 및 설명

## 배경

- **타워 디펜스 게임**은 `D:\vs\ChipGames\ChipTowerGame` 프로젝트로 이관되었습니다.
- 기존 ChipGameRoot 저장소는 **애드센스가 연결된 메인 도메인** (https://chipgames.github.io)을 유지합니다.
- 이 프로젝트는 단일 게임이 아닌 **게임 목록/허브 페이지**로 전환됩니다.

## 목표

1. ChipGames GitHub Pages 메인 사이트를 **게임 목록 페이지**로 전환
2. 각 게임별 **썸네일 이미지**를 활용한 이미지 기반 게임 목록 표시
3. 이미지 클릭 시 해당 게임 페이지로 이동
4. **라이트/다크 모드** 지원
5. **다국어** 지원 (한국어, 영어, 중국어, 일본어)
6. **Google AdSense** 유지·적용
7. **SEO** 메타 태그, 구조화 데이터, 시맨틱 마크업 적용
