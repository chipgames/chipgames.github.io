# 작업 단계 (체크리스트)

## 기본 구성
- [ ] **1단계**: `delete` 폴더 내 불필요한 파일 삭제
- [ ] **2단계**: 각 게임별 썸네일 이미지 추가 (`images/games/`)
- [ ] **3단계**: 새 `index.html` 생성 — 이미지 기반 게임 목록 카드/그리드 UI
- [ ] **4단계**: `style.css` — 게임 목록 페이지용 스타일, **라이트/다크 모드** 테마

## 기능 적용
- [ ] **5단계**: **다국어** — `lang/` 번역 파일 및 언어 선택기
- [ ] **6단계**: **AdSense** 스크립트 적용 (Ads.txt 유지)
- [ ] **7단계**: **SEO** — Meta 태그, Open Graph, Twitter Card, Schema.org (ItemList)

## 정리 및 배포
- [ ] **8단계**: `robots.txt`, `sitemap.xml` 갱신
- [ ] **9단계**: 법적 페이지 (`privacy.html`, `terms.html`, `contact.html`) 유지/수정
- [ ] **10단계**: PWA 설정 (`sw.js`, `site.webmanifest`) 간소화
- [ ] **11단계**: GitHub Actions 배포 확인 및 https://chipgames.github.io 최종 검증
